# Number Puzzle Game 🎮
A console-based puzzle game built in Java.

## How to play
- Choose difficulty: Easy / Medium / Hard
- Solve the puzzle shown on screen
- You have limited attempts based on difficulty

## Features
- Addition puzzles
- Mixed operator puzzles (with BODMAS logic)
- 3 difficulty levels

## How to run
javac Game.java
java Game
